Presupuestos Topografía Chile — PresupuestoServicios
=================================================

Proyecto: App para calcular y controlar presupuestos de trabajos topográficos (Google Sheets + AppSheet).

Contenido creado:

- `docs/GoogleSheets_structure.md` — estructura de tablas/hojas y columnas.
- `docs/Formulas_y_Expresiones.md` — fórmulas de Google Sheets y expresiones AppSheet.
- `docs/Vistas_Automatizaciones.md` — vistas AppSheet y automatizaciones sugeridas.
- `data/` — CSV con datos de ejemplo para cada tabla.
- `docs/checklist_implementacion.md` — pasos para desplegar la app.
- `scripts/crear_presupuesto_sheets.gs` — script de Apps Script para generar el Spreadsheet.
- `scripts/IMPLEMENTACION_APPSHEET_PDF.md` — instrucciones para PDF automatizado.

Siguiente paso recomendado:

1. Crear un Google Sheet por cada hoja listada en `docs/GoogleSheets_structure.md` y pegar los encabezados.
2. Copiar los CSV desde `data/` o importarlos al Google Sheet para probar fórmulas.
3. Conectar el Google Sheet a AppSheet y crear las vistas indicadas en `docs/Vistas_Automatizaciones.md`.

Si quieres, yo puedo:
- Generar el Google Sheet en tu Google Drive (requiere permisos).
- Crear las expresiones AppSheet paso a paso.
