ChasConChile - AppSheet para topografía urbana
=============================================

Este repositorio se ha reorganizado para que la implementación activa de AppSheet y Google Sheets esté centralizada en la carpeta `appsheet/`.

Estructura principal:

- `appsheet/README.md` — visión general del proyecto AppSheet.
- `appsheet/docs/` — documentación de diseño y guías de implementación.
- `appsheet/scripts/` — scripts de Google Apps Script para crear hojas y carpetas.
- `appsheet/data/` — datos de ejemplo y catálogos.
- `appsheet/img/` — recursos gráficos.
- `appsheet/outputs/` — plantillas y archivos de ejemplo.

Para iniciar el trabajo, abre `appsheet/README.md`.
