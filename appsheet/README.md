RTK CORS Gestión Topográfica
============================

Proyecto de inicio para construir la app "RTK CORS Gestión Topográfica" en AppSheet.

Objetivo:
- Configurar desde cero con control de versiones, documentación mínima y estructura ordenada.
- Preparar el proyecto para GitHub y desarrollo ágil de una app no-code/low-code orientada a topografía urbana en Chile.

Estructura del proyecto:
- `README.md` — visión general del proyecto.
- `.gitignore` — exclusiones de Git.
- `CHANGELOG.md` — historial de cambios.
- `docs/BITACORA.md` — bitácora de trabajo.
- `docs/ROADMAP.md` — roadmap del proyecto.
- `docs/DECISIONES.md` — decisiones arquitectónicas y de diseño.
- `docs/APP_SHEET_DISEÑO.md` — diseño unificado del AppSheet.
- `scripts/crear_topografia_sheets.gs` — Apps Script para generar Google Sheets y carpetas Drive.

Módulos principales:
- Clientes
- Proyectos
- Tipos de Servicio
- Presupuestos
- Pre-Campo
- Salida a Campo
- Post-Proceso
- Entregables
- Documentos automáticos
- Automatizaciones y dashboards
