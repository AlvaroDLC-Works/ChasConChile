# Roadmap — RTK CORS Gestión Topográfica

## Fase 1 — MVP inicial
- Configurar Google Sheets base de datos.
- Crear app AppSheet mínima con módulos:
  - Clientes
  - Proyectos
  - Tipos de Servicio
  - Presupuestos
  - Pre-Campo
  - Salida a Campo
- Definir estados y flujos principales.
- Crear automatización básica de proyecto y carpeta Drive.
- Generar primer documento PDF de propuesta/presupuesto.

## Fase 2 — Flujo completo y automatizaciones
- Agregar Post-Proceso y Entregables.
- Crear plantillas Google Docs: Informe técnico, Propuesta, Acta de entrega.
- Automatizar generación de PDF y guardado en Drive.
- Mejorar dashboards operativos y financieros.
- Registrar bitácora y fotos en terreno.

## Fase 3 — Version Pro
- Crear roles y permisos multusuario.
- Integrar con Apps Script para carpetas Drive automáticas.
- Añadir validaciones avanzadas y reportes mensuales.
- Refactorizar para futura migración a base de datos robusta.
