# Guía de seguimiento diario

Usa este formato para registrar el avance diario del proyecto.

## Formato
- Fecha:
- Qué se hizo:
- Qué falta:
- Errores:
- Próximos pasos:

## Ejemplo
- Fecha: 2026-06-21
- Qué se hizo: Se creó la estructura inicial del proyecto y los documentos base.
- Qué falta: Definir las tablas de AppSheet y configurar los cálculos de presupuesto.
- Errores: Ninguno.
- Próximos pasos: Crear el modelo de datos en Google Sheets.
